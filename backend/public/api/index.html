glossolalia (API documentation)

glossolalia-backend.herokuapp.com

All queries must be encoded (e.g., c++ turns into c%2B%2B). You may use this tool.

programming-language

Endpoint that returns data about programming languages.

Endpoints

/api/programming-languages

Description

Get all programming languages from database.

Parameters

name type multiple return example
id integer false n == 1 ?id=5
name string false n == 1 ?name=php
first_appeared integer false n >= 0 ?first_appeared=1995
paradigm string[] true; n >= 0; ?paradigm=functional,imperative
designed_by string[] true; n >= 0; ?designed_by=Rasmus%20Lerdorf
typing_discipline string[] true; n >= 0; ?typing_discipline=strong,dynamic
influenced_by string[] true; n >= 0; ?influenced_by=perl,c%2B%2B
influenced string[] true; n >= 0; ?influenced=java,c%23
website_url; string[] true; n >= 0; ?website_url=https%3A%2F%2Fwww.scala-lang.org%2F
source_url string[] true; n >= 0; ?source_url=https%3A%2F%2Fen.wikipedia.org%2Fwiki%2FHaskell_(programming_language)
filename_extension string[] true; n >= 0; ?filename_extension=.c%2C.h
  • Some parameters accepts multiple values, provided with comma separated strings.
  • If no query is entered, all programming languages are fetch.

Example

endpoint returns
/api/programming-languages all programming languages
/api/programming-languages/?id=11 a single programming language of ID 11
/api/programming-languages/?name=perl a single programming language of name perl
/api/programming-languages/?first_appeared=2001 all programming languages first appeared in 2001
/api/programming-languages/?paradigm=concurrent,functional all programming languages that have both concurrent and functional paradigm
/api/programming-languages/?designed_by=Ryan all programming languages designed by someone named Ryan
/api/programming-languages/?typing_discipline=static, inferred all programming languages that have both static and inferred typing discipline
/api/programming-languages/?influenced_by=c,java all programming languages influenced by c and java
/api/programming-languages/?influenced=rust,go all programming languages that influenced rust and go
/api/programming-languages/?website_url=https%3A%2F%2Fwww.perl.org%2F all programming languages that have the entered url as website
/api/programming-languages/?source_url=https%3A%2F%2Fen.wikipedia.org%2Fwiki%2FPHP all programming languages that have the entered url as source
/api/programming-languages/?filename_extension=.php%2C.phar all programming languages that have .php and .phar as filename extension

keyword

Endpoint that returns data about programming languages keywords.

Endpoints

/api/keywords

Description

Get all programming languages keywords from database.

Parameters

name type multiple return example
id integer false n == 1 ?id=8
programming_language_id integer false n >= 0 ?programming_language_id=3
name string false n >= 0 ?name=enum
